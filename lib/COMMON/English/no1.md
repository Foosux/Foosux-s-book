# 时态

## I am doing

现在进行时：表示事情的延续，还未结束。

句形： `am/is/are`+`-ing`

```
I am      (=I'm)  | driving
he/she/it is (=he's) | working
we/you/they are (=we're) | doing  
```

> `I'm doing something`
表示正处于做某事的中间环节，还未结束。并不仅仅是此时此刻在进行的事。

## I do

一般现在时：形容普遍的事情，它一直在发生或者循环往复，如：太阳东升西落。

句形：`drive(s)/work(s)/do(es)`

```js
// 第一、二人称
I/we/you/they drive/work/do

// 第三人称加 s(es)
he/she/it     dirves/works/does  
```

> 第三人称单数后的谓语（动词）要变形

常用 `do/does` 来提问或者否定
