# vue生命周期(8)

![](https://cn.vuejs.org/images/lifecycle.png)

> 生命周期钩子的 `this` 上下文指向调用它的 Vue 实例。   

> 不要在选项属性或回调上使用箭头函数 如：   
> created: () => console.log(this.a)
> vm.$watch('a', newValue => this.myMethod())

<!-- toc -->

## demo

```js
new Vue({
  data: {
    a: 1
  },
  created: function () {
    // `this` 指向 vm 实例
    console.log('a is: ' + this.a)
  }
})
// => "a is: 1"
```
