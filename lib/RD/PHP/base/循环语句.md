# PHP循环语句
<!-- toc -->

## while

满足条件时执行循环。

```js
<?php
  while (条件) {
    // 要执行的代码;
  }
?>
```

## do...while

循环体至少执行一次。

```js
<?php
  do {
    // 要执行的代码;
  }
  while (条件);
?>
```

## for

可指定循环的次数。

```js
<?php
  for (初始值; 条件; 增量) {
    // 要执行的代码;
  }
?>
```

## foreach

用于遍历数组。

```js
<?php
  foreach ($array as $value) {
    // 要执行代码;
  }
?>
```
